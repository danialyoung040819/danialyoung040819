# LaTeX Rhythm Refiner

Use this skill when a LaTeX manuscript is technically correct but reads as monotonous, over-formulaic, or AI-like. Improve readability without changing claims, citation semantics, math, labels, cross-references, or table/figure references.

## Rules

1. Preserve every citation command, label, reference, equation, and environment.
2. Do not add new claims or citations.
3. Vary sentence length and paragraph rhythm.
4. Remove filler transitions and inflated significance language.
5. Keep terminology consistent with the existing manuscript.
6. Return a concise change summary and any sentences that still need author verification.
