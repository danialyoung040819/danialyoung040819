# SART Clinic Data Access

Use this skill for tasks involving public SART clinic data lookup, clinic PKID identification, and selecting the correct SART report page for a requested metric.

## Workflow

1. Search public SART sources first rather than guessing clinic identifiers.
2. Verify the clinic identity and PKID against public pages.
3. Open the relevant SART report page for the requested metric.
4. Record exact source URLs and the metric/report context.
5. Distinguish public data from assumptions or unavailable private data.
